# 离线识别热词

热词只作用于 `POST /v1/transcribe` 的离线结果，不修改 X-ASR 实时流式 partial/final。模型管理的“离线识别热词”面板保存后，下一次离线识别立即生效；后端每次识别都会检查文件内容变化，无需重启。

## hot.txt

每行格式：

```text
标准词|别名1|别名2~~~黑名单1|黑名单2
```

示例：

```text
撒贝宁|撒贝你|撒贝林~~~撒贝宁工作室
```

先执行别名精确替换，再按拼音相似度处理近音结果。出现该条目的黑名单文本时，本条热词不会替换。

## hot-rule.txt

每行格式为 `正则表达式 = 替换文本`，无效正则会被忽略：

```text
50赫兹 = 50Hz
第([一二三四五])章 = Chapter \1
```

## API

- `GET /v1/hotwords`：读取当前配置和两个文本文件。
- `PUT /v1/hotwords`：保存开关、阈值、热词和规则。
- `POST /v1/hotwords/preview`：预览一段文字的替换结果和命中明细。

数据默认保存在 `data/asr/hotwords/`。可通过 `ASRAPP_HOTWORDS_DIR` 修改目录。
