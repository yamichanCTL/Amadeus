# 启动桌面前端计划

> **父文档**: [← 返回文档索引](../README.md)

## 任务目标

- 在已配置后端地址 `http://127.0.0.1:8000` 的前提下启动桌面 Electron 前端。
- 确认 Vite 开发服务器监听 `5173`，并验证前端页面可访问。

## 影响范围

- 运行 `frontend/desktop` 开发服务器和 Electron 客户端。
- 不修改应用业务代码。

## 实现步骤

1. 检查 8000、5173 端口与现有进程。
2. 运行 `npm run dev`。
3. 验证 Vite 页面和后端健康接口。

## 风险评估

- WSL/Electron GUI 依赖 DISPLAY/WSLg；若 Electron 无法显示，保留 Vite Web 页面作为回退。
- 后端当前未通过健康探测，前端可能先显示断开状态。
